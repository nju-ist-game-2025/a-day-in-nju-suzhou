# UI 使用指南

## 已实现功能

### 1. 主菜单 (MainMenu)
- **开始游戏** 按钮：点击进入游戏场景
- **退出游戏** 按钮：点击关闭应用程序
- 简洁的界面设计，带有标题和按钮样式

### 2. 游戏视图 (GameView)
- 800x600 的游戏场景
- 可移动的玩家角色
- 基本的场景边界
- 按 **ESC** 键返回主菜单
- 支持加载PNG图片资源

### 3. 玩家控制
- **方向键 ↑↓←→**：移动角色
- **WASD 键**：发射子弹（W上、S下、A左、D右）
- **可以同时移动和射击**！
- **ESC 键**：返回主菜单
- 射击冷却：150毫秒（可调节）

## 文件结构

### UI 相关文件
- `src/ui/mainmenu.h/cpp` - 主菜单界面
- `src/ui/gameview.h/cpp` - 游戏场景视图

## 🎨 如何添加美工资源

### 目录结构

在游戏可执行文件（game_final.exe）所在目录创建 `resources` 文件夹：

```
build/
├── game_final.exe
└── resources/
    ├── player.png        # 玩家图片
    ├── bullet.png        # 子弹图片
    ├── background.png    # 背景图片
    └── enemy.png         # 敌人图片
```

### 资源规格要求

#### 玩家图片 (player.png)
- **推荐尺寸**：50x50 像素（或更大，会自动缩放）
- **格式**：PNG（支持透明背景）
- **说明**：代表玩家角色的图片

#### 子弹图片 (bullet.png)
- **推荐尺寸**：50x50 像素
- **格式**：PNG（支持透明背景）
- **说明**：玩家发射的子弹图片

#### 背景图片 (background.png)
- **推荐尺寸**：800x600 像素
- **格式**：PNG 或 JPG
- **说明**：游戏场景的背景图

#### 敌人图片 (enemy.png)
- **推荐尺寸**：40x40 像素
- **格式**：PNG（支持透明背景）
- **说明**：敌人角色图片（预留）

## 编译和运行

```bash
cd build
cmake ../our_game
cmake --build .
./game_final.exe
```

## 注意事项

- 所有图片资源建议使用 **PNG格式**（支持透明）
- 图片会自动缩放到指定大小
- 相对路径是相对于 **可执行文件** 的位置
- 资源加载失败会显示警告信息，方便调试
